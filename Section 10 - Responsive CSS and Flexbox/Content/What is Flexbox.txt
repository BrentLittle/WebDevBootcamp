a one-dimensional layout for laying out items in rows or columns

distribute space dynamically across elements of an unknown size.

Two Axis
Main Axis - Left to Right
Cross Axis - Top to Bottom

flex-direction lets us decide if we want a row or column
Default is row which is left to right

Justify Content
determines how the content is distributed across the main Axis


Flex-Wrap
determines if elements will wrap across multiple lines

Align items
determines how the content is distributed across the cross Axis

Align Content & Align-self
used to distribute space if there are multiple rows or columns depending on axis


Flex-basis
defines the initial size of an element before additional space is distributed

Flex Grow
Controls amount of space an element takes up.
